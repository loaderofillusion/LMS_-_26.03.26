Оператор if: принятие решений

В реальной жизни мы постоянно принимаем решения: «Если идет дождь, я возьму зонт, иначе — не возьму». В программировании для этого используется оператор if (если). Он позволяет выполнять разные блоки кода в зависимости от условий.

Простой if

Самый простой оператор if проверяет условие и, если оно истинно (True), выполняет код внутри блока.

age = 18

if age >= 18:
    print('Вы можете голосовать!')
    print('И покупать билеты в кино для взрослых')

print('Программа продолжается')

Важные правила:

  • После if ставится условие, затем двоеточие :
  • Код, который выполняется при истинном условии, пишется с отступом (обычно 4 пробела или Tab)
  • Как только отступ заканчивается, Python понимает, что блок закончился

Операторы сравнения

В условиях мы используем операторы сравнения. Они возвращают True или False.

ОператорЗначениеПримерРезультат
==равно5 == 5True
!=не равно5 != 3True
>больше10 > 5True
<меньше3 < 7True
>=больше или равно5 >= 5True
<=меньше или равно4 <= 6True
⚠️ Важно! Не путайте = (присваивание) и == (сравнение). age = 18 — присваивает значение, age == 18 — проверяет, равно ли 18.

Операторы else и elif

Часто нужно предусмотреть, что делать, если условие не выполнилось. Для этого есть else (иначе). А если нужно проверить несколько вариантов — elif (сокращение от else if).

Конструкция if-else

temperature = 25

if temperature > 30:
    print('Очень жарко! Включите кондиционер.')
else:
    print('Температура комфортная.')

Конструкция if-elif-else

score = 85

if score >= 90:
    grade = 'Отлично'
    print('Твой результат:', grade, '⭐')
elif score >= 70:
    grade = 'Хорошо'
    print('Твой результат:', grade, '👍')
elif score >= 50:
    grade = 'Удовлетворительно'
    print('Твой результат:', grade, '😐')
else:
    grade = 'Нужно подтянуть знания'
    print('Твой результат:', grade, '📚')

Важно: elif проверяется только если предыдущие условия были ложны. Как только находится истинное условие, остальные пропускаются.

Логические операторы

Иногда нужно проверить несколько условий одновременно. Для этого используются логические операторы:

  • and — "и" (оба условия должны быть истинны)
  • or — "или" (хотя бы одно условие истинно)
  • not — "не" (меняет значение на противоположное)
age = 15
has_parent_permission = True

# Проверяем несколько условий
if age >= 18 or (age >= 14 and has_parent_permission):
    print('Можно зарегистрироваться')
else:
    print('Нужно разрешение родителей')

# Использование not
is_weekend = False
if not is_weekend:
    print('Сегодня рабочий день')

# Сложное условие
weather = 'солнечно'
temperature = 22

if (weather == 'солнечно' or weather == 'облачно') and 18 <= temperature <= 25:
    print('Идеальный день для прогулки!')

Проверка на вхождение

Оператор in позволяет проверить, содержится ли элемент в коллекции:

# Проверка символа в строке
letter = 'а'
text = 'программирование'

if letter in text:
    print(f'Буква "{letter}" есть в слове "{text}"')

# Проверка значения в списке
favorite_color = 'синий'
colors = ['красный', 'синий', 'зеленый']

if favorite_color in colors:
    print(f'{favorite_color} есть в списке любимых цветов')
🎮 Игра "Угадай число"

Пример использования условных операторов:

secret_number = 7
guess = int(input('Угадай число от 1 до 10: '))

if guess == secret_number:
    print('Поздравляю! Ты угадал! 🎉')
elif guess > secret_number:
    print('Загаданное число меньше.')
else:
    print('Загаданное число больше.')

Вложенные условия

Условия могут быть вложенными — одно условие внутри другого.

has_ticket = True
has_passport = False
age = 16

if has_ticket:
    print('У вас есть билет')
    if has_passport:
        print('Паспорт есть, можно проходить')
    else:
        print('Нет паспорта! Вход запрещен')
else:
    print('Сначала купите билет')

Тернарный оператор

Для простых условий можно использовать сокращенную запись:

# Обычная запись
age = 17
if age >= 18:
    status = 'взрослый'
else:
    status = 'ребенок'

# Тернарный оператор (условие? если True : если False)
status = 'взрослый' if age >= 18 else 'ребенок'
print(f'Статус: {status}')

Практические примеры

# Калькулятор скидок
total_price = 1500
if total_price > 1000:
    discount = 0.1
    final_price = total_price * (1 - discount)
    print(f'Скидка 10%! Итоговая сумма: {final_price} руб.')
else:
    print(f'Сумма к оплате: {total_price} руб.')

# Определение времени года
month = 6
if month in [12, 1, 2]:
    season = 'зима'
elif month in [3, 4, 5]:
    season = 'весна'
elif month in [6, 7, 8]:
    season = 'лето'
else:
    season = 'осень'
print(f'{month} месяц — это {season}')
