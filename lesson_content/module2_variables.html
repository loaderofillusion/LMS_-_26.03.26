Что такое переменные?

Представьте, что у вас есть коробки разных размеров. В каждую коробку вы можете положить что-то одно: игрушку, книгу, конфету. Вы даже можете наклеить на коробку наклейку с названием того, что внутри. В программировании переменные — это такие "коробки" для хранения данных.

Переменная — это именованное место в памяти компьютера, где хранится какая-то информация. У переменной есть:

  • Имя — как наклейка на коробке
  • Значение — то, что лежит внутри
  • Тип данных — вид того, что внутри (число, текст и т.д.)

Создаем переменные

В Python переменная создается, когда вы присваиваете ей значение с помощью знака = (это оператор присваивания).

# Создаем переменные
name = 'Анна'        # текст (строка)
age = 10             # целое число
height = 1.45        # дробное число
is_student = True    # логическое значение (True/False)

# Используем переменные
print('Меня зовут', name)
print('Мне', age, 'лет')
print('Мой рост', height, 'м')

Обратите внимание: знак = в программировании означает не "равно", а "присвоить значение". Читается как: "переменной name присвоить значение 'Анна'".

Правильные имена переменных

Имена переменных должны быть:

  • Понятными — student_name лучше, чем sn
  • Начинаться с буквы или подчеркивания — name, _private
  • Содержать только буквы, цифры и подчеркивания
  • Не совпадать с ключевыми словами Python — например, нельзя назвать переменную print или if
📝 Правильные имена:

user_name, score2, _max_value, studentAge

❌ Неправильные имена:

2score (начинается с цифры), user-name (дефис не разрешен), print (ключевое слово)

Числовые типы данных

В Python есть два основных числовых типа:

int — целые числа

Это числа без дробной части: 5, -10, 0, 1000. С ними можно выполнять все арифметические операции.

a = 10
b = 3

print(a + b)   # 13 (сложение)
print(a - b)   # 7  (вычитание)
print(a * b)   # 30 (умножение)
print(a / b)   # 3.333... (деление — всегда дает float)
print(a // b)  # 3  (целочисленное деление)
print(a % b)   # 1  (остаток от деления)
print(a ** b)  # 1000 (возведение в степень)

float — числа с плавающей точкой

Это числа с дробной частью: 3.14, -0.5, 2.0. В Python десятичная часть отделяется точкой, а не запятой!

pi = 3.14159
radius = 5.0
area = pi * radius ** 2
print('Площадь круга:', area)  # 78.53975
💡 Важно! Когда вы делите целые числа с помощью /, результат всегда становится float (дробным числом). Если нужен целый результат, используйте //.

Строки и булевы значения

str — строки (текст)

Строки — это последовательности символов. Они всегда заключаются в кавычки. Строки можно складывать (конкатенировать) и умножать на число.

first_name = 'Анна'
last_name = 'Иванова'
full_name = first_name + ' ' + last_name  # 'Анна Иванова'

# Умножение строк
laugh = 'ха' * 5  # 'хахахахаха'

# Длина строки
print(len(full_name))  # 12 (считает все символы, включая пробел)

# Доступ к символам (нумерация с 0)
print(full_name[0])  # 'А'
print(full_name[5])  # 'И'

Строки можно создавать разными способами:

# Три способа записать строку
message1 = 'Привет'
message2 = "Привет"
message3 = """Длинный текст
который может занимать
несколько строк"""

bool — булевы значения

Булев тип может принимать только два значения: True (истина) и False (ложь). Они используются для проверки условий.

is_adult = True
has_license = False

# Булевы значения часто получаются в результате сравнения
age = 15
can_vote = age >= 18  # False, так как 15 не больше или равно 18
is_teen = 13 <= age <= 19  # True, возраст от 13 до 19

Преобразование типов

Иногда нужно преобразовать данные из одного типа в другой. Для этого есть специальные функции:

# Преобразование в строку
number = 42
text = str(number)  # '42'

# Преобразование в число
age_str = '15'
age_int = int(age_str)  # 15
pi_str = '3.14'
pi_float = float(pi_str)  # 3.14

# Важно: нельзя преобразовать в число то, что не является числом
# int('привет') — вызовет ошибку!

Практический пример

# Программа-калькулятор возраста
birth_year = 2010
current_year = 2024
age = current_year - birth_year

name = input('Как вас зовут? ')  # input() запрашивает ввод от пользователя
print(f'Привет, {name}! Тебе {age} лет или скоро будет.')

# input() всегда возвращает строку, поэтому если нужен возраст — преобразуйте
user_age = int(input('Сколько вам лет? '))
if user_age >= 18:
    print('Вы уже взрослый!')
else:
    print('Вы еще учитесь в школе!')
